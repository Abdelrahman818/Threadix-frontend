# Threadix

Threadix is an e-commerce platform that sells clothes, cups, and custom prints.

This repository contains both the frontend and backend of the application.

## Project Structure

- `frontend/`: Next.js based frontend application.
- `backend/`: Node.js/Express based backend API.

## Getting Started

### Frontend
Navigate to the `frontend` directory:
```bash
cd frontend
npm install
npm run dev
```

### Backend
Navigate to the `backend` directory:
```bash
cd backend
npm install
npm start
```
